## 免费在线桌游

### 平台 (Platforms)

1. **Board Game Arena** (https://boardgamearena.com)
   - 免费在线桌游平台
   - 包含200+款经典桌游
   - 支持实时多人对战

2. **Tabletopia** (https://tabletopia.com)
   - 免费版提供多款桌游
   - 真实3D桌游体验
   - 支持跨平台游玩

3. **Yucata** (https://www.yucata.de)
   - 完全免费的桌游网站
   - 德式桌游为主
   - 无需付费即可游玩所有游戏

## 打印即玩资源

### 网站推荐 (Recommended Sites)

1. **BoardGameGeek - PnP Section**
   - 网址: https://boardgamegeek.com
   - 搜索 "Print and Play"
   - 大量免费设计师作品

2. **PnP Arcade**
   - 专门的打印即玩游戏网站
   - 定期更新新游戏
   - 完全免费下载

3. **Button Shy Games - Free PnP**
   - 提供免费的打印即玩游戏
   - 高质量设计
   - 适合家庭游玩

### 打印建议 (Printing Tips)

- 使用彩色打印获得更好体验
- 建议使用厚卡纸（200-300g）
- 可以过塑保护卡牌
- 使用硬币、小石子作为配件

## DIY自制桌游

### 基础材料 (Basic Materials)

- 卡纸/厚纸板
- 色笔/彩色打印
- 骰子（可用在线骰子代替）
- 棋子（可用硬币、纽扣、小石子等）
- 塑封膜（可选，用于保护）

### 简单自制游戏推荐

1. **自制卡坦岛 (DIY Catan)**
   - 六边形瓦片可以用纸板剪裁
   - 资源卡可以手绘
   - 骰子使用标准六面骰

2. **自制狼人杀 (DIY Werewolf)**
   - 仅需纸牌
   - 手写角色名称
   - 完全免费

3. **自制象棋/围棋 (DIY Chess/Go)**
   - 画出棋盘
   - 用不同颜色的纽扣或硬币作为棋子

## 数字版桌游

### 免费/低价数字桌游

1. **Steam 平台**
   - 等待促销打折（通常折扣很大）
   - 关注 Humble Bundle 慈善包
   - 免费周末活动

2. **手机应用**
   - 很多桌游有免费的手机版
   - 支持AI对战或在线对战
   - 如: Monopoly, Carcassonne, Ticket to Ride 等都有免费试玩版

3. **网页版桌游**
   - lichess.org (国际象棋)
   - playok.com (多种棋类游戏)
   - 完全免费，无需下载

# DIY 桌游制作指南 (DIY Board Game Guide)

## 为什么要自己制作桌游？

- 💰 **经济实惠**: 成本低至几元
- 🎨 **个性定制**: 完全按自己喜好设计
- 🧠 **学习体验**: 理解游戏机制
- 👨‍👩‍👧‍👦 **家庭活动**: 与家人共同制作

## 简单DIY项目

### 项目1: 自制UNO牌

**难度**: ⭐☆☆☆☆
**时间**: 2-3小时
**成本**: ¥5-10

**材料**:
- 卡纸（可以用旧名片或硬纸）
- 彩色笔（红、黄、绿、蓝、黑）
- 剪刀
- 尺子

**步骤**:

1. **准备卡牌**
   - 剪裁108张 6cm x 9cm 的卡片
   - 保持尺寸一致很重要

2. **设计卡面**
   ```
   数字牌 (每种颜色各19张):
   - 0 × 1
   - 1-9 × 2 (每个数字2张)
   
   功能牌 (每种颜色各2张):
   - Skip (跳过) × 2
   - Reverse (反转) × 2  
   - Draw Two (+2) × 2
   
   万能牌:
   - Wild (变色) × 4
   - Wild Draw Four (+4变色) × 4
   ```

3. **绘制**
   - 用彩色笔在卡片上写数字和符号
   - 可以添加装饰图案
   - 背面画统一图案

4. **保护**
   - 过塑（推荐）
   - 或用透明胶带包边

**游戏规则**: 网上搜索 "UNO规则" 即可

### 项目2: 自制象棋/围棋

**难度**: ⭐⭐☆☆☆
**时间**: 3-4小时  
**成本**: ¥10-15

**材料**:
- 大张纸板或布
- 尺子、铅笔
- 黑色马克笔
- 硬币或纽扣（32个不同颜色/标记）

**步骤（象棋）**:

1. **制作棋盘**
   - 画一个9×10的格子
   - 每格约4cm×4cm
   - 添加楚河汉界
   - 标记炮和将的位置

2. **制作棋子**
   - 用硬纸板剪32个圆形
   - 或使用32个硬币/纽扣
   - 用两种颜色区分红黑方
   - 写上棋子名称（帅、仕、相、马、车、炮、兵）

3. **规则**
   - 下载打印象棋规则
   - 或在线学习

### 项目3: 自制卡坦岛

**难度**: ⭐⭐⭐⭐☆
**时间**: 8-10小时
**成本**: ¥30-50

**材料**:
- 厚纸板
- 彩色纸或颜料
- 小木块或硬币（道路和城市）
- 骰子 2个
- 卡纸（资源卡）

**简化版步骤**:

1. **制作地形板块**
   - 剪19个六边形（边长约5cm）
   - 分别涂色：
     * 树林（深绿）× 4
     * 牧场（浅绿）× 4  
     * 农田（黄色）× 4
     * 山丘（灰色）× 3
     * 砖地（红色）× 3
     * 沙漠（米色）× 1
   - 在每个板块上写数字标记（2-12）

2. **制作资源卡**
   - 每种资源各20张卡片
   - 木材、羊毛、小麦、砖、矿石
   - 可以手绘图标

3. **制作道路和城市标记**
   - 道路：小木条或画线
   - 村庄：小方块
   - 城市：大方块

4. **简化规则**
   - 下载并打印基础规则
   - 或观看视频教程

**提示**: 卡坦岛规则较复杂，建议先玩在线版熟悉后再制作

### 项目4: 自制狼人杀

**难度**: ⭐☆☆☆☆  
**时间**: 1小时
**成本**: ¥3-5

**材料**:
- 卡纸或名片大小的硬纸
- 笔

**步骤**:

1. **确定人数和角色**
   ```
   9人局推荐配置:
   - 狼人 × 3
   - 村民 × 3
   - 预言家 × 1
   - 女巫 × 1
   - 猎人 × 1
   ```

2. **制作身份牌**
   - 剪9张大小一致的卡片
   - 在每张卡片上写角色名
   - 背面画统一图案

3. **可选增强**
   - 为每个角色画图标
   - 添加技能说明文字
   - 过塑保护

4. **规则**
   - 非常简单，网上搜索即可
   - 需要一个主持人

### 项目5: 自制飞行棋

**难度**: ⭐⭐☆☆☆
**时间**: 2-3小时
**成本**: ¥5-10

**材料**:
- 大纸板（40cm × 40cm）
- 彩色笔（红、黄、蓝、绿）
- 骰子
- 16个棋子（可用纽扣、硬币等，4种颜色各4个）

**步骤**:

1. **绘制棋盘**
   - 画十字形轨道
   - 每条臂有13格
   - 中间有大十字交叉区
   - 四个角各有一个飞机场（起点）

2. **标记特殊格子**
   - 起飞点
   - 快速通道
   - 终点

3. **准备棋子**
   - 4种颜色各4个
   - 可用不同颜色的纽扣

4. **规则**
   - 经典飞行棋规则
   - 掷骰子前进

## 制作技巧

### 1. 整齐划线
```
- 使用尺子和铅笔打底
- 确认无误后用黑笔描线
- 擦除铅笔痕迹
```

### 2. 配色建议
```
- 使用对比色便于区分
- 避免太多颜色造成混乱
- 参考原版游戏的配色
```

### 3. 卡牌大小标准
```
- 扑克牌: 6.3cm × 8.8cm
- 桌游标准: 6cm × 9cm  
- 可以略大一些方便书写
```

### 4. 提高耐用性
```
- 过塑最有效（强烈推荐）
- 用厚卡纸（200g以上）
- 边缘用透明胶带加固
- 避免锐角，剪圆角
```

### 5. 收纳建议
```
- 用小盒子分类存放
- 卡牌用橡皮筋捆绑
- 标注游戏名称和件数
```

## 创意改造

### 主题定制

将现有游戏改成你喜欢的主题：

**例如**: 狼人杀改成三国杀风格
- 狼人 → 叛军  
- 村民 → 忠臣
- 预言家 → 诸葛亮
- 女巫 → 华佗

### 简化版本

制作经典游戏的简化版：

- **简化卡坦**: 减少板块和资源种类
- **简化大富翁**: 减少格子数量，简化规则
- **简化三国杀**: 只用基础武将和简单技能

## 从零设计自己的游戏

### 步骤1: 确定核心机制

选择一个或组合几个：
- 掷骰子前进
- 卡牌收集
- 区域控制  
- 工人放置
- 猜谜/问答

### 步骤2: 设计原型

- 用纸笔快速画出来
- 不要追求美观
- 专注于可玩性

### 步骤3: 测试

- 自己先玩几遍
- 邀请朋友测试
- 记录问题和改进点

### 步骤4: 迭代

- 根据反馈修改规则
- 调整平衡性
- 简化复杂部分

### 步骤5: 制作正式版

- 美化设计
- 制作最终版本
- 编写清晰的规则书
